# Xade Extra

Userscript con funciones extra para la plataforma "xade" de la Xunta de Galicia. 

Funciones disponibles actualmente:
- Lista de usuarios: Dentro de "Alumnado -> Xestión de alumnado", al final de la lista de alumnos aparece un botón "Lista completa" que permite copiar fácilmente los nombres de los alumnos en distintos formatos.

## 🚀 Instalación

1. Instala un gestor de userscripts:
    - Greasemonkey (Firefox)
    - Tampermonkey (Chrome / Edge)

2. Haz clic en el siguiente botón:

[![Instalar Userscript](https://img.shields.io/badge/Instalar-Userscript-brightgreen)](https://raw.githubusercontent.com/AlvaroBeiro/xade-extra/main/xade-beautifier.user.js)

3. Acepta la instalación en el navegador.
